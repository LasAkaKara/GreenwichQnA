#GreenQ&A Student Forum Website

#Installation
1. Install XAMPP, starting APACHE and MySQL
3. Extract basecode files to ../xampp/htdocs (depends on your installation)
2. Run localhost/phpMyAdmin on browser
3. Create a new database named 'greenqa'
4. Import the Database.sql file in the zip file and run
5. Run localhost/GreenQ&A/login.php on browser
